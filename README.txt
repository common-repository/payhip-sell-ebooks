== Installation ==

This section describes how to install the plugin and get it working.

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'payhip'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `payhip-sell-digital-downloads.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `payhip-sell-digital-downloads.zip`
2. Extract the `payhip-sell-digital-downloads` directory to your computer
3. Upload the `payhip-sell-digital-downloads` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard

== Frequently Asked Questions ==

= What is Payhip? =
Payhip is an e-commerce platform that enables anyone to Sell Downloads directly to their fans and followers.

= How can my customers pay? =
We currently support Paypal and Stripe. So your customers can use their Paypal account or their credit card to purchase.

= Are my downloads safe with Payhip? =
Your files are safe with us, we have stringent security in place so that your customers will only get your digital downloads only if they have paid for them. In addition, the download link will expire and can not be used by anyone else.

= How secure is Payhip? =
Payhip is secure. We don't store card details - all payments are handled by Paypal and Stripe.

= How much do you charge for this service? =
There are no set-up, monthly or hosting fees. You only pay a small 5% transaction fee after paypal has taken its standard fee, this means that you only pay when you get paid. For more information see pricing.

= What digital download formats do you support? =
We support ALL file formats. Including MP4, MP3, EPUB, PDF, MOBI and many more.

= Can I sell on Payhip if I sell on other platforms like Amazon? =
Yes. You can sell on Payhip even if you sell downloads elsewhere. Payhip is here to help you market your digital downloads better.

= How can I contact you? =
Just shoot us an email at contact@payhip.com or reach out to us on Twitter.
